# FTP
FTP客户端和服务器在Python，项目为IS301在SJTU。
# #代码结构
' ' '<Br>
|——GUI  <Br>
|------ ClientGui.py pyqt5中的客户端GUI<Br>
|------ loggui .py登录界面<Br>
|——图标客户端GUI图标<Br>
登录、下载和上传等对话框<Br>
|——utils.py获取文件信息<Br>
|——Client .py FTP客户端<Br>
|——Server .py FTP服务器<Br>
' ' '<Br>
# #特性
1. 用户可以访问ftp服务器。<Br>
2. 客户端GUI可以显示本地和远程文件列表。<Br>
3. 用户可以下载和上传文件。<Br>
4. 用户可以在本地创建、删除、重命名文件<Br>
